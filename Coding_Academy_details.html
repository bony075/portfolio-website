Coding Academy

  1. => It's a programming language related website where contains navbar that is courses, blog, FAQ.
  2. => There is also remain three login that is Email login, Google login, GitHub login. If a user want to see details of the course then he/ she must register with the system and log in with it.
  3. => For registration user needs some basic information like his/ her full name, image, email address and a password then submit register.
  4. => After login with the system user see his/ her profile picture on the right side top. There is also a toggle button for dark or light theme.
  5. => User select course from the left side but without log in user can't see the details of the course.
  6. => User also download pdf of the course details from the download button.
  7. => If a user get a premium access then system direct him/ her a payment method.
  8. => Firebase authentication used for user information.
